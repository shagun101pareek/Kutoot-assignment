# Simple API

A basic REST API built using Node.js and Express.js.

## Features

- GET endpoint
- POST endpoint
- JSON responses
- Basic error handling

---

## Tech Stack

- Node.js
- Express.js
- JavaScript

---

## Endpoints

### GET /users

Returns the list of users.

Example response:

```json
{
  "success": true,
  "data": []
}
